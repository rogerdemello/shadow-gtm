# Shadow GTM — 90-Second Demo Script

A tight, judge-facing run. Goal: land the **one loop** —
`live web → AI reasoning → a play a rep can run today` — and end on the War Room.

**Before you record:** if a live demo with keys, run one scan first so signals are
cached, then refresh. Otherwise, hit **✦ Try with demo data** for instant state.
Use `ANTHROPIC_MODEL=claude-sonnet-4-6` for snappier live streaming.

### Backups / fallbacks
- **No keys / offline?** Hit *✦ Try with demo data* — the dashboard fills instantly
  with curated signals + a pre-baked battlecard. The streaming routes will report
  a clean "key not set" error if you trigger them, which is itself a graceful UX
  moment.
- **Scan feels slow on stage?** Demo-data the feed first, then trigger one live
  scan for the streaming-in effect.
- **JS-heavy angle:** mention the **⚡JS** badge — those sites (e.g. LinkedIn)
  render through Bright Data's **Scraping Browser**, a third BD product in play.
- **Evidence skeptic:** click two signals in a row to drive home that *every*
  one carries the source quote — this is the moat over generic "AI competitive
  intelligence" tools.
